# Tutorial: Building bootable system with BootC

Welcome! This repository contains the source code and instructions for building your own Operating System using Container runtime and BootC

By the end of this tutorial, you will:
1.  Define an OS using a `Containerfile`.
2.  Build your OS image and convert to different format using bootc image builder
3.  Run it as a virtual machine.
4.  Update the Containerfile with a patch fix
5.  Update the machine directly from patched image
6.  Rollback to previous version
...etc.
